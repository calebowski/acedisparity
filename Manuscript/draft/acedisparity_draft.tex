\begin{abstract}
\noindent
Ancestral state estimation is a widely used method to infer unobserved traits across a phylogeny, extending analyses beyond incomplete fossil records. 
In studies of morphological disparity ancestral state estimation can account for ghost lineages, yet estimates are often confounded with high uncertainty. 
How best to deal with this uncertainty remains unclear. 
Using simulations, we compared pre- and post-ordination estimation methods, as well as both point estimate and probabilistic approaches for treating uncertainty, under continuous and discrete trait models and varying fossil sampling levels.
The relative disparity error was calculated from the difference between our estimated disparity values and the true disparity values from the original simulations.
We then extended this framework to evaluate the recovery of changes in disparity across time by simulating mass extinction events with varying intensity and selectivity.
We find that pre-ordination probabilistic estimations outperform post-ordination and point estimation methods and that omission of ancestral state estimations leads to underestimates of disparity.
As expected, higher fossil sampling improves estimation accuracy, although recovered disparity accuracy was robust even at low fossil sampling. 
Similar patterns were observed when recovering changes in disparity across mass extinction events but declined significantly when fossil sampling was low, highlighting that fossil inclusion is essential when assessing disparity in deep time.
These results demonstrate that explicitly propagating uncertainty, rather than collapsing states into point estimates, is more likely to recover accurate patterns in disparity, reinforcing the need to move beyond point estimates when working with the incomplete fossil record.
\end{abstract}

\noindent\textbf{Keywords:} ancestral state estimation; morphological disparity; fossils; simulation; uncertainty; macroevolution

\subsection{Ancestral state estimation}
Ancestral state estimation was carried out on the down-sampled matrices using the \texttt{multi.ace} function in \texttt{dispRity}, which is a wrapper for \texttt{ace} in the \texttt{ape} package \citep{paradis2019ape}.
We used four combinations of two approaches to ancestral state estimation to evaluate each method's accuracy in recovering the true disparity signal.
\subsubsection{Pre-ordination estimation} \label{pre-ord}
Firstly, we compared pre- versus post-ordination estimation methods. 
Pre-ordination estimation methods of discrete characters involves estimating the marginal likelihoods for each discrete state at each node, based on a Maximum-Likelihood derived estimate of the M\textit{k} transition matrix and the tree topology given \citep{pagel1999maximum,yang2006computational}.
Given that the transition rates were set \textit{a priori} during simulations, the equal-rates (ER) model was used to avoid model misspecification as a confounding factor on estimation accuracy \citep{keating2023best}.
%cs: could do an analysis where ARD model is used to see if results change with model mispecification (Russell mentioned this)? 

Pre-ordination estimation of continuous characters models evolution under a Brownian motion model, whereby ancestral state values lie within a multivariate normal distribution that is estimated given the observed tip data and the shared evolutionary history between taxa, i.e., the tree topology \citep{schluter1997likelihood,rohlf2001comparative}. 
Confidence intervals were calculated from standard errors derived from the Hessian matrix at the maximum likelihood estimate. % CS: not sure if this is necessary but put it in since the confidence interval will be mentioned lots in rest of paper.
The width of confidence intervals reflects the uncertainty in continuous ancestral estimates, which typically increases with node depth and decreases with the number of informative descendant lineages.
For a recent review of the methods described here see \cite{revell2025ancestral}.

\subsubsection{Post-ordination estimation} \label{post-ord}
Post-ordination estimation involves the estimation of a node's position within a multidimensional ordination matrix (or trait space).
The generation of an ordination matrix differed between discrete and continuous characters.
For discrete characters, distance matrices were computed using the `MORD' metric \citep{lloyd2016estimating} and ordinated using principal coordinates analyses (PCoA) (\texttt{cmdscale()} function).
Continuous data were ordinated using principal components analysis (PCA) (\texttt{prcomp()} function).
For both the discrete- and continuous- character derived trait spaces, the ancestral states across all PC axes were then estimated under a Brownian motion model, as detailed in section \ref{pre-ord}.

\subsubsection{Uncertainty treatment} \label{uncertainty}
For each estimation method, we also treated uncertainty in estimates with both deterministic and probabilistic approaches.
The deterministic approach involves collapsing the likelihood outputs into point estimates.
For discrete characters, this involved a strict majority rule whereby the state with the highest scaled likelihood was selected at each node for each character.
In scenarios where state likelihoods are equally tied, one state is collapsed at random.
We also employed a relative majority rule, whereby any state at a scaled likelihood greater than the maximum observed scaled likelihood minus the inverse number of possible states is collapsed, otherwise states are left as uncertain.
For continuous characters, the deterministic approach used the median of the estimated ancestral confidence interval. 

The probabilistic approach differs in that uncertainty in estimates are explicitly accounted for through the replication of matrices.
For instance, discrete states at each node for each character were `sampled' according to their scaled likelihoods across 100 matrix replications.
In a scenario where there is uncertainty in a binary character for a specific node, e.g., 0.55 for state 0 and 0.45 for state 1, this probabilistic approach would have a 55\% probability of sampling state 0 in each matrix for the respective node.
Continuous characters were treated similarly, with 100 matrices replicated and ancestral state estimates for each node and character sampled across a normal distribution within their respective estimated confidence interval.
Both procedures are implemented in the \texttt{dispRity} package using the \texttt{sample = 100} argument in the \texttt{multi.ace} function \citep{guillerme2018disprity}.

\subsection{Disparity metrics}
To ensure fair comparison between different fossil sampling levels only the nodes present across all fossil sampling levels were retained from the ancestral state estimation outputs for each phylogeny; therefore, comparisons in disparity calculations between fossil sampling levels differed solely on the difference in number of fossil tips and the estimations of node states.
Discrete and continuous pre-ordination ancestral state estimations were ordinated in the same manner as described in section \ref{post-ord}.

When assessing overall disparity, we wanted to understand how well ancestral state estimation can recover true patterns in size and density in trait space \citep{guillerme2020shifting}.
Since we were assessing overall disparity, position was not a metric that can be discerned due to the centring of ordinated matrices.
Size-based metrics describe the spread of points in the trait space \citep{guillerme2020shifting}.
To explore how well the size of the trait space was recovered, we calculated the sum of quantiles.
The sum of quantiles calculates the range on each PC axis for the 95\% of the data and then sums these ranges across all axes.
This metric has similar properties to the often used sum of ranges metric by describing the total trait space exploited \citep{foote1992rarefaction}, but with less sensitivity to outliers.
For density, we measured the sum of variances \citep{ciampaglio2001detecting,wills2001morphological} and the average nearest neighbour distance \citep{foote1992rarefaction}.
The sum of variances is the most widely used metric in disparity analyses.
Disparity values were calculated across each estimation and the `true' simulation so that a relative disparity error could be calculated.

\begin{equation}
\text{Relative disparity error} = \frac{\text{Estimated disparity} - \text{True disparity}}{\text{True disparity}}
\end{equation}

Disparity calculations were carried out across three treatments of the ordinated matrices.
Firstly, the untreated ordination output was used.
Secondly, we min-max standardised the PC axes so that the absolute magnitude in PC scores did not influence the disparity metric scores.
Thirdly, we only used the minimum number of dimensions that were contained across all fossil sampling levels for that specific phylogeny.
For the simulations of 50 living taxa trees, this would be the first 48 dimensions, in the 100 living taxa it would be 98 dimensions etc.

\subsection{Mass extinction simulations}
Having established the accuracy of different ancestral state estimation methods in recovering overall trait space patterns, we next tested how well these methods capture changes in disparity across time during simulated mass extinction events.
Following the framework above, 100 trees were simulated for each tree size of 50, 100 and 150 living taxa with a mass extinction event.
We simulated both random, where taxa were eliminated at non-selectively, and selective extinctions, where taxa were more prone to extinction based on their trait values.
Extinction severity was set with a uniform distribution between 75--95\%, the range seen between K-Pg and the Permian extinctions.

With the new set of trees (and trait data for selective extinctions), the framework from section \ref{trait simulation} to \ref{uncertainty} was repeated. 
Rather than evaluating the overall disparity estimate, we assessed how well the methods could recover the relative disparity change across the extinction event.
Two bins that were equidistant based off the total tree height divided by six were sampled either side of the extinction event.
Disparity was measured from the taxa contained within each of these two bins to calculate a relative disparity change across the extinction event.

\begin{equation}
\text{Relative change in disparity} = \frac{\text{Disparity post-extinction} - \text{Disparity pre-extinction}}{\text{Disparity pre-extinction}}
\end{equation}
