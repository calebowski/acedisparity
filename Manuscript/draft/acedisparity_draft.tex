\begin{abstract}
\noindent
Ancestral state estimation is a widely used method to infer unobserved traits across a phylogeny, extending analyses beyond incomplete fossil records. 
In studies of morphological disparity ancestral state estimation can account for ghost lineages, yet estimates are often confounded with high uncertainty. 
How best to deal with this uncertainty remains unclear. 
Using simulations, we compared pre- and post-ordination estimation methods, as well as both point estimate and probabilistic approaches for treating uncertainty, under continuous and discrete trait models and varying fossil sampling levels.
The relative disparity error was calculated from the difference between our estimated disparity values and the true disparity values from the original simulations.
We then extended this framework to evaluate the recovery of changes in disparity across time by simulating mass extinction events with varying intensity and selectivity.
We find that pre-ordination probabilistic estimations outperform post-ordination and point estimation methods and that omission of ancestral state estimations leads to underestimates of disparity.
As expected, higher fossil sampling improves estimation accuracy, although recovered disparity accuracy was robust even at low fossil sampling. 
Similar patterns were observed when recovering changes in disparity across mass extinction events but declined significantly when fossil sampling was low, highlighting that fossil inclusion is essential when assessing disparity in deep time.
These results demonstrate that explicitly propagating uncertainty, rather than collapsing states into point estimates, is more likely to recover accurate patterns in disparity, reinforcing the need to move beyond point estimates when working with the incomplete fossil record.
\end{abstract}

\noindent\textbf{Keywords:} ancestral state estimation; morphological disparity; fossils; simulation; uncertainty; macroevolution

\subsubsection{Post-ordination estimation} \label{post-ord}
Post-ordination estimation involves the estimation of a node's position within a multidimensional ordinated matrix (commonly referred to as the trait space, morphospace, functional space, etc.).
The generation of an ordinated matrix differed between discrete and continuous characters.
For discrete characters, distance matrices were computed using the `MORD' metric \citep{lloyd2016estimating} %TG: probably worth at least spelling out the acronym for a quick understanding for the reader.
 and ordinated using principal coordinates analyses (PCoA) (\texttt{cmdscale()} function). 
Continuous data were ordinated using principal components analysis (PCA) (\texttt{prcomp()} function).
For both the discrete- and continuous- character derived trait spaces, the ancestral states across all PC axes were then estimated under a Brownian motion model, as detailed in section \ref{pre-ord}.

For each estimation method, we also treated uncertainty in estimates with both deterministic and probabilistic approaches. %TG: not sure what you're referring to with deterministic and probabilistic here. I think you mean more "point estimate" and "distribution" no?
The deterministic approach involves collapsing the likelihood outputs into point estimates. %TG: Re: comment about, yeah, I don't think deterministic means this in this context (I think it means the outcome can be reliably predicted or something like that).
For discrete characters, this involved a strict majority rule whereby the state with the highest scaled likelihood was selected at each node for each character.

The probabilistic approach differs in that uncertainty in estimates are explicitly accounted for by sampling multiple results rather than a single point estimate. %TG: (changed the replication thing here - more like a sampling); also, probablistic approach here should be more called something like the "distribution method/approach".
For each discrete character states at a node the states where sampled a 100 times according to their scaled likelihoods (resulting in a 100 character estimation matrices). %TG: rephrased a bit here. Mainly sampled didn't need quotes ;).
In a scenario where there is uncertainty in a binary character for a specific node, e.g., 0.55 for state 0 and 0.45 for state 1, this distribution approach would results in a 55\% probability of sampling state 0 in each matrix for the respective node (and 45\% chance of sampling state 1).
